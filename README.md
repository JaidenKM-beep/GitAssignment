# Carnival Mystery Game

## Description
Carnival Mystery is a C++ text-based adventure game where the player explores a carnival to solve the mystery of the stolen lucky gold ring. The player visits game booths, food stands, and rides to collect clues, earn tokens, and uncover who stole the ring and where they are hiding.

## Features
- Explore different carnival locations
- Visit game booths to win tokens
- Use tokens at food stands
- Ride attractions that may reveal clues or events
- Random chance events using C++
- Player location tracking using pointers
- Uses STL vector and map
- Mystery storyline with clues and ending
